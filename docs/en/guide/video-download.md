# Video Download

Download videos from YouTube and Bilibili with built-in yt-dlp.

## ✅ Supported Formats

| Platform     | Type       | Example                        |
| ------------ | ---------- | ------------------------------ |
| **YouTube**  | Standard   | `youtube.com/watch?v=xxx`      |
|              | Short URL  | `youtu.be/xxx`                 |
|              | Shorts     | `youtube.com/shorts/xxx`       |
|              | Embed      | `youtube.com/embed/xxx`        |
| **Bilibili** | BV/av      | `bilibili.com/video/BVxxx`     |
|              | Multi-part | `bilibili.com/video/BVxxx?p=2` |
|              | Short URL  | `b23.tv/xxx`                   |

## ❌ Not Supported

| Platform | Type              | Reason            |
| -------- | ----------------- | ----------------- |
| YouTube  | Playlists/Channel | Use single video  |
| Bilibili | Anime/Movies      | Copyright         |
|          | Paid courses      | Purchase required |
|          | Livestream        | Real-time         |
|          | Premium content   | Requires cookies  |
