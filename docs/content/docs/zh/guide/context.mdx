## 🎯 传统机翻的问题

普通翻译工具是 **逐句翻译**：

```
输入: "He's gone."
输出: "他走了。"
```

但这句话在不同场景下意思完全不同：
- 葬礼上 → "他去世了。"
- 追逐戏 → "他跑了！"
- 分手场景 → "他离开我了。"

**没有上下文，就没有准确翻译。**

---

## 🧠 MioSub 的解决方案

### 长上下文窗口

MioSub 使用 **Gemini** 的超长上下文窗口（100K+ tokens），可以：

- 一次性读取 **整个视频** 的字幕
- 理解故事背景、人物关系、情节发展
- 根据语境选择最合适的译法

### 翻译流程

```
┌─────────────────────────────────────┐
│  1. 读取全文                         │
│     AI 先通读所有字幕，理解整体内容    │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  2. 语义分段                         │
│     按场景/话题切分为 5-10 分钟片段   │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  3. 上下文翻译                       │
│     每个片段翻译时携带前后文信息      │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│  4. 润色校对                         │
│     检查前后一致性，修正译名          │
└─────────────────────────────────────┘
```

## 🎭 效果对比

| 场景 | 逐句机翻 | MioSub 上下文翻译 |
| :--- | :--- | :--- |
| 葬礼 | "He's gone." → "他走了" | "他已经离开了我们" |
| 追逐 | "He's gone." → "他走了" | "他跑了！" |
| 俚语 | "Break a leg!" → "摔断腿" | "祝你好运！" |
| 双关 | 完全丢失 | 尽可能保留或注释 |

## 💡 使用技巧

### 润色功能

如果某段翻译不满意：
1. 选中该片段
2. 点击 **"润色翻译"**
3. AI 会根据上下文重新翻译

### 批量重新生成

如果需要整体调整：
1. 选中多个片段
2. 点击 **"重新生成"**
3. 将重新执行完整流程
