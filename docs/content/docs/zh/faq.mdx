## 🔑 API 相关

### API 报错 429 怎么办？

**错误含义**：请求频率超过了 API 限制。

**解决方案**：
1. **等待几分钟**：免费额度每分钟 15 次，等待后自动恢复
2. **添加多个 Key**：在设置中用逗号分隔多个 API Key，系统会自动轮询
3. **降低并发**：在设置中降低翻译并发数

---

### 提示 "API Key 无效" 怎么办？

1. 确认 Key 是否完整复制（以 `AIza` 开头）
2. 确认 Key 前后没有多余的空格
3. 尝试在 [AI Studio](https://aistudio.google.com/app/apikey) 重新生成一个

---

### 提示 "地区不支持"？

Gemini API 在部分地区受限。请确保网络环境可以正常访问 Google 服务。

---

## 🎬 视频处理

### 下载视频失败怎么办？

**常见原因**：
- **网络问题**：确保能正常访问 YouTube/Bilibili
- **链接格式错误**：使用标准视频链接，不支持播放列表
- **版权限制**：B站番剧、付费内容无法下载

**解决方案**：
1. 检查网络连接
2. 使用单个视频链接（非播放列表）
3. 尝试更新 yt-dlp（设置中可更新）

---

### 处理速度很慢？

**可能原因**：
- 使用的是 CPU 版 Whisper，速度较慢
- 网络延迟导致 API 调用缓慢

**优化建议**：
1. 使用较小的 Whisper 模型（Base 或 Small）
2. [配置 GPU 加速](/docs/guide/whisper)
3. 使用 Whisper API 而非本地模型

---

### FFmpeg 找不到怎么办？

MioSub 已内置 FFmpeg，通常不需要额外配置。

如果仍然报错：
1. 确认使用的是完整版安装包（不是精简版）
2. 检查 `resources` 文件夹下是否有 `ffmpeg.exe`
3. 尝试重新下载完整安装包

---

## 📝 字幕相关

### 字幕时间轴不准确？

**解决方案**：
1. 启用 [强制对齐功能](/docs/guide/alignment) 获得毫秒级精度
2. 在编辑器中手动微调时间轴
3. 检查是否源视频音频有问题

---

### 翻译质量不理想？

**优化建议**：
1. **选择正确的场景预设**：动漫、电影、科技等不同风格
2. **使用术语表**：确保专有名词翻译一致
3. **尝试润色功能**：选中字幕后使用"润色翻译"

---

### Windows 下出现乱码？

**可能原因**：字体不支持或编码问题

**解决方案**：
1. 导出时选择 UTF-8 编码
2. 在字幕样式中选择支持中文的字体（如微软雅黑）
3. 检查系统语言设置

---

## 🖥️ 软件使用

### 软件打不开/闪退？

1. 确认系统是 Windows 10 或更高版本
2. 尝试以管理员身份运行
3. 检查是否被杀毒软件误拦截
4. 重新下载完整安装包

---

### 如何更新软件？

MioSub 支持自动更新。如果自动更新失败：
1. 访问 [Releases 页面](https://github.com/corvo007/Gemini-Subtitle-Pro/releases)
2. 下载最新版本
3. 直接覆盖旧版本即可
